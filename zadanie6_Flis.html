zagadnienia teoretyczne
  1. Rodzaje okienek
  2. Okienko alert – wyświetlająca okienko dialogowe z informacją i przyciskiem OK,
  3. Okienko prompt – generująca okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.
  4. Okienko confirm służy do tego, aby użytkownik coś potwierdził (stąd nazwa confirm).
  1. Konwersja liczb
    Wczytanie zmiennej z użyciem prompt wykonywane jest jako ciąg znaków dlatego konieczna jest konwersji z ciągu znaków na wartości liczbowe aby wykonywac obliczenia.
  2. parseFloat(zmienna) - konwersja zmiennej do typu float
  3. parseInt(zmienna) - konwersja zmiennej do typu int (liczbowego)
  4. toString(zmienna) - konwersja zmiennej do typu łańcuchowego
  1. Przy nadawaniu nazw zmiennym należy przestrzegać pewnych zasad
  2. 1. Nazwa zmiennej musi się zaczynać od litery, znaku "$" lub "_", może zawierać tylko litery (bez polskich znaków), cyfry oraz "$" i "_". Poprawne nazwy zmiennych: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz Błędne nazwy zmiennych: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, wysokość, imie+nazwisko.
  3. 2. Wielkość liter ma znaczenie. Rekord i rekord to dwie różne zmienne! Jest to bardzo częsta przyczyna błędów w kodzie początkujących programistów.
  4. 3. Należy nie używać słów kluczowych JavaScript jako nazw zmiennych.

     W JavaScript nie ma możliwości definiowania różnych typów zmiennych tak jak w C++, typy zmiennych są definiowane w trakcie ich przypisywania. Podstawowe zmienne liczbowe w JavaScript to:
  2. Number --> Typ liczbowy

  4. zmiennoprzecinkowe
  5. String --> Typ tekstowy
  6. Boolea --> Typ logiczny
